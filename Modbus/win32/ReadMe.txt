========================================================================
    動態連結程式庫：Modbus 專案概觀
========================================================================

AppWizard 已經為您建立了這個 Modbus DLL。  

這個檔案的內容摘要，包含各個構成 Modbus 應用程式的檔案。


Modbus.vcproj
    這是使用應用程式精靈所產生之 VC++ 專案的主要專案檔。 
    它含有產生該檔案之 Visual C++ 的版本資訊，以及有關使用應用程式精靈
    所選取之平台、組態和專案功能的資訊。

Modbus.cpp
    這是主要 DLL 原始程式檔。

	這個 DLL 建立之後，不會匯出任何符號。因此，它
	在建置時不會產生 .lib 檔。如果您希望這個專案
	成為其他專案的相依專案，您必須
	加入程式碼從 DLL 匯出某些符號，以便產生
	匯出程式庫，或者，您可以在專案的 [屬性頁] 對話方塊中，
	將 [連結器] 資料夾 [一般] 屬性頁上的 Ignore Input Library 屬性
	設定為 Yes。
